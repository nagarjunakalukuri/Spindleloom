### NEW binding constraints the fixes exposed
1. **Handoff filenames are an unspecified contract.** The context pack keys the estimation→sprint edge on `09-estimates.md`; estimation-facilitator wrote `09-estimation.md`. The richest downstream handoff in the run dropped silently, and only manifested as a 3-vs-5-sprint contradiction. A1 connected the deep edges; this is the next absent one promoted to binding — the run-2 pattern ("each repaired edge promotes the next") repeating at the last hop.

No check reconciles "PBIs referenced" against "PBIs defined with AC." The fleet (estimation + sprint-planner) caught it; the tooling did not.
3. **The flag register surfaces but does not close.** A5 mechanized the re-work queue; there is still no re-dispatch that sends backlog-manager back to author the missing PBI-PLAT-004 row or product-analytics back to cut the metric PBI. Feedback is now *visible and owned*, but loop-closure remains manual.
