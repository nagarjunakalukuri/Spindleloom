# Windsurf bundle

Generated by Shipwright. Copy `.windsurf/` and `mcp/` into your repo root. Windsurf has no subagents, so each role is a `model_decision` rule (activates when relevant); `000-spindleloom-conventions.md` is always-on. Rules cap at ~12k chars — split a long one if Windsurf truncates it.

## Live traceability (MCP)

Windsurf's MCP config is **user-global**, so add this to `~/.codeium/windsurf/mcp_config.json` (install [uv](https://docs.astral.sh/uv/) — it provisions the MCP SDK on first run), with absolute paths to where you copied `mcp/`:

```json
{
  "mcpServers": {
    "spindleloom": {
      "command": "uv",
      "args": [
        "run",
        "--with",
        "mcp[cli]",
        "python",
        "/abs/path/to/your/repo/mcp/mcp_server.py"
      ],
      "env": {
        "SPINDLELOOM_SPEC_ROOT": "/abs/path/to/your/repo"
      }
    }
  }
}
```
