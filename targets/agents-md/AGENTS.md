This project carries a fleet of role-specialist AI agents spanning the SDLC (market → spec → design → build → test → ship → operate). When a task matches a role below, **adopt that agent's instructions** — full text in `agents/<name>.md`. Run only the subset your team needs (start with `doc-strategy-advisor`). Conventions: Req-ID `<DOC>-<AREA>-<NUM>`, one living RTM, "the system shall …" (ISO/IEC/IEEE 29148); see `knowledge_hub/BEST-PRACTICES.md`.
