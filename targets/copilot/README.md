# GitHub Copilot bundle

Generated by `build_harness_artifacts.py`. Copy `.github/`, `.vscode/`, and `mcp/` into your repo root. Chat modes + `copilot-instructions.md` carry the fleet; `.github/prompts/` carries the slash commands; `.claude/skills/` is auto-read by Copilot; `.vscode/mcp.json` registers the live traceability server, launched via `uv run` (install [uv](https://docs.astral.sh/uv/); it provisions the MCP SDK on first run).
