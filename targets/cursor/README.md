# Cursor bundle

Generated by `build_harness_artifacts.py`. Copy `.cursor/`, `.claude/`, and `mcp/` into your repo root. Each role is a native Cursor agent (`.cursor/agents/`); slash commands live in `.cursor/commands/`; skills in `.claude/skills/` (Cursor auto-reads them); `000-spindleloom-conventions.mdc` is the always-on rule. `.cursor/mcp.json` registers the live traceability server, launched via `uv run` (install [uv](https://docs.astral.sh/uv/); it provisions the MCP SDK on first run); it uses `${workspaceFolder}`, so it's portable across machines.
